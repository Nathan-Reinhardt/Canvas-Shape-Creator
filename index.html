How To

Draw

Welcome, you won't be able to draw right away so here are some

steps to help you draw some drawings on this canvas. First you want to choose a drawing. At the top on the left it will show "Drawing #1" selected meaning that the first drawing has been selected to work with. Next step is to choose a shape to draw with the current drawing. Below selecting the drawing number is the shape picker. Once you pick a shape it will tell you the current shape selected. Now to draw you click, drag, and unclick to draw shapes of any size. To add more attributes is located below the shape picker.

The max amount of drawings you can have on the canvas at one

time is three of them. To switch drawings, you can click on "Drawing #2" or "Drawing #3" to work with an entirely different drawing. The last feature about the draw feature is when you want to clear a drawing. The red clear button at the very bottom on the left hand side once clicked will clear every single attribute from your current drawing to be set back to default and to clear it off of the canvas. Just beware you can lose all the work you do with the clear button which is why it is red. Now onto Save and Load feature.

Save and Load

The last attribute property works with more than one drawing.

The Save and Load feature works as saving artworks and loading artworks. First the save feature will take every single attribute for every single drawing on the canvas and save them later when you want to load the previous work you saved. Once saved and you change anything with the current canvas, it will not effect the current save. The load feature will replace all current drawings with the saved values saved from earlier. Warning, the save and load feature persists for only one save state.

The clear button also works with the save and load feature.

If you save 3 drawings and after saving you clear two of the three drawings, once loaded the 3 old drawings will come back. When you clear than save it will know that any drawing that was cleared and not worked on again will not be drawn. You don't only have to save artworks. Just drawing one drawing and saving also works. Lots of logic and edgecasing went into having the clear feature work with save and load. In the future I would want to have more than one save state for artworks.
